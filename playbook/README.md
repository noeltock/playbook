# playbook/

This folder is the source of truth for this project's goals, tool doctrine, design rules, decisions, and errors. It is read by agents on every session.

| File | Purpose |
|------|---------|
| `NORTH-STAR.md` | Project goals and success criteria |
| `TOOLS.md` | Tool-use rules for agents |
| `DESIGN.md` | Product and UX rules (optional) |
| `STYLEGUIDE.md` | Visual rules (optional) |
| `tokens.json` | DTCG design tokens (optional) |
| `DECISIONS.md` | Rendered decision log |
| `ERRORS.md` | Rendered error/warning log |
| `decisions.jsonl` | Append-only ADR source |
| `errors.jsonl` | Append-only error source |
| `skills/` | `/playbook-*` skills |
| `hooks/` | Claude Code PostToolUse hooks |
| `viewer/` | Fumadocs docs site (optional) |
| `schemas/` | JSON Schemas for validation |
